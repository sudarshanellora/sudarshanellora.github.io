My Projects

Image Classification

Collaborated with 6 peers to post-process and extract statistical information such as mean, median from 2481 Computed Tomography (CT) images to classify for Covid or Non-Covid conditions, Analyzed the images by extracting Gray-Level Co-occurrence Matrix (GLCM) features, performed classification using Support Vector Machine (SVM) and Random Forest models giving an accuracy of 91% and 94.5% respectively, Published this work at Emitter International Journal of Engineering Technology, June 2022.

Multi Class Image Classification

Collaborated with 6 individuals and extracted image features and gathered image features for 21,164 images which belongs to imbalanced and improper dataset, Programmed to Classify the X-Ray images based on its classes such as Covid, Lung Opacity, Normal, Viral Pneumonia using Residual neural network (ResNet), U-Net, Convolution neural network, Mobile V2 Net with an accuracy of 70.13%, 93.30%, 95.42% and 89.75% respectively.

Survillance Robot

Built a mobile robot using 2 DC motors, motor driver circuit and Raspberry pi 3B+ model, Designed the mobile robot to move forward, backward and sideways whilst capturing the live data from Logitech camera mounted on the robot, Programmed using OpenCV to navigate and capture live streaming simultaneously with time delay less than 1sec.

Health Monitoring System

Lead a team of 4 to design a health monitoring system using raspberry pi and took input data samples using max 30102 high sensitivity Pulse Oximeter sensor, Generated heart rate by peak-to-peak sampling and regressed a deep neural network model for blood pressure using mimic 3 dataset with 20.1115% loss.
